# Code of Conduct

We follow the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

## In one paragraph

Be respectful. Assume good faith. Focus on the technical substance.
Harassment, personal attacks, discrimination, and doxxing are not
welcome — they'll get a warning, then a ban.

## Enforcement

Report incidents privately via the same channel as [SECURITY.md](SECURITY.md).
The maintainer decides on the action (warning, temporary ban, permanent
ban) within 7 days. Appeals are welcome; the decision after appeal is
final.

## Scope

Applies to GitHub issues, PRs, discussions, the DHN chat (if we ever
add one), and any public representation of the project (talks, blog
posts, related repos).
